# Panel Administrativo — Parcial 2 Prog IV

Panel web para la gestión de un sistema de pedidos de restaurante. Permite administrar productos, categorías, ingredientes, pedidos (con tablero Kanban por estado) y roles de usuarios.

Construido con React, TypeScript, Tailwind CSS, React Query y Zustand.

## Requisitos

- [Node.js](https://nodejs.org/) v18+
- [pnpm](https://pnpm.io/)

## Instalación y uso

```bash
# 1. Clonar el repositorio
git clone <url-del-repositorio>
cd parcial2_repo_admin

# 2. Instalar dependencias
pnpm install

# 3. Levantar en modo desarrollo
pnpm run dev
```
